# React Pizza v2 üçï

A pizza ordering web app built with React, SCSS Modules, and React Router.  
This project allows users to browse pizza categories, add items to the cart, and simulate orders using mock data.

## Technologies Used
- **React** (hooks, functional components)
- **SCSS Modules** for styling
- **React Router** for navigation
- **MockAPI** for fake backend data

## Features
- Browse different pizza categories
- Add/remove pizzas to/from the cart
- Simulate orders with mock API
- Responsive UI for desktop and mobile
